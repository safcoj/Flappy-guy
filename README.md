# Flappy guy

## Описание

Это простая аркадная игра, вдохновлённая классическим форматом «tap-to-fly», разработанная на Unity.

## Требования

Для корректной работы проекта необходимо установить:

* **Unity Editor версии 6000.4.0f1**

## Установка и запуск

1. Склонируйте или скачайте данный репозиторий.
2. Откройте **Unity Hub**.
3. Добавьте проект через кнопку **Add**.
4. Убедитесь, что используется версия Unity **6000.4.0f1**.
5. Откройте сцену:

   ```
   MainScreen
   ```
6. Нажмите **Play** для запуска игры.

## Управление

* **Левый клик мыши**
* **Пробел (Space)**

Оба действия заставляют персонажа подпрыгивать.

## Примечание

Если сцена не запускается корректно, убедитесь, что все сцены добавлены в **Build Settings**.
